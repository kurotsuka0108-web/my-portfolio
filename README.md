# 下園 司 - ポートフォリオサイト

フルスタックエンジニアを目指す下園 司のポートフォリオサイトです。

## 技術スタック

- **フレームワーク**: Next.js 16 (App Router)
- **UI**: React 19 + TypeScript
- **スタイリング**: Tailwind CSS v4
- **アニメーション**: Framer Motion
- **アイコン**: Lucide React

## セクション構成

| セクション | 内容 |
|---|---|
| Hero | 自己紹介・プロフィールイラスト |
| BentoGrid | 作品・スキル・趣味・連絡先をベントボックス風に表示 |
| Contact | お問い合わせフォーム |

## ローカルで起動する

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

http://localhost:3000 でアクセスできます。

## ビルド

```bash
npm run build
npm run start
```
